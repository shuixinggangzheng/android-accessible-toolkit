# 隐私保护策略

## 概述

本项目严格遵循隐私保护原则，所有用户数据均在设备本地处理，不上传任何云端服务。

## 三大核心原则

### 1. 本地处理

- **语音识别**：使用 Vosk 本地模型，所有语音数据在设备本地处理
- **人声检测**：基于本地 AudioRecord 实时分析，无网络传输
- **文本转语音**：使用 Android 系统 TTS，完全本地化
- **数据存储**：仅使用内存缓存，不写入本地数据库

### 2. 即时丢弃

- **转写结果**：仅在内存中保留，应用关闭后立即删除
- **会话历史**：仅保留本次会话数据，结束会话后清空
- **音频数据**：实时处理后立即丢弃，不保存录音文件
- **无日志记录**：不记录任何用户语音或文本内容

### 3. 用户可控

- **功能开关**：每个功能模块可独立启用/禁用
- **权限管理**：用户可随时撤销任何权限
- **服务控制**：用户可随时启动/停止无障碍服务
- **数据清除**：提供一键清除所有本地数据功能

## 数据安全

### 本地存储

- **不使用数据库**：所有数据仅在内存中处理
- **不写入文件**：不将用户数据保存到本地文件
- **缓存清理**：应用退出时自动清理所有缓存

### 网络安全

- **无云端服务**：不连接任何外部服务器
- **局域网通信**：跨设备功能仅在局域网内工作
- **无数据上传**：不向任何第三方上传用户数据

### 第三方依赖

- **Vosk SDK**：开源语音识别库，仅本地使用
- **Java-WebSocket**：开源 WebSocket 库，仅局域网通信
- **AndroidX**：Google 官方库，无数据收集

## 用户权利

### 知情权

- 用户可随时查看隐私政策
- 应用启动时明确说明数据处理方式
- 提供详细的权限用途说明

### 控制权

- 用户可随时启用/禁用任何功能
- 用户可随时撤销任何权限
- 用户可随时停止服务并清除数据

### 删除权

- 应用卸载时自动删除所有数据
- 用户可手动清除所有本地缓存
- 无云端数据，无需远程删除

## 合规声明

- 遵循 GDPR（通用数据保护条例）
- 遵循 CCPA（加州消费者隐私法案）
- 遵循中国《个人信息保护法》
- 不收集任何个人身份信息（PII）

## 技术实现

### 隐私保护措施

```kotlin
// 1. 本地 Vosk 模型
val model = Model(modelPath) // 本地加载，无网络传输

// 2. 实时处理，不保存
audioRecord.read(buffer, 0, size) // 实时读取
processBuffer(buffer) // 立即处理
// buffer 随后被覆盖，无持久化

// 3. 内存缓存，会话结束清除
class SubtitleViewModel {
    private val history = mutableListOf<TranscriptResult>()
    
    fun clearHistory() {
        history.clear() // 会话结束时清除
    }
}
```

1. **定期清理**：建议用户定期清除应用缓存
2. **权限审查**：定期检查并撤销不必要的权限
3. **更新应用**：及时更新应用以获取最新安全补丁
4. **谨慎授权**：仅授予必要权限
